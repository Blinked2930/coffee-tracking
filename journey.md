# User Journey: Kafe Tracker

## 1. Onboarding
- **Step 1:** User opens the PWA link on their mobile device.
- **Step 2:** User is presented with a simple login screen.
- **Step 3:** User enters their pre-assigned Name and 1-8 digit PIN.
- **Step 4:** User is authenticated and lands on the Home Screen.

## 2. Logging a Kafe (The Main Action)
- **Step 1:** On the Home Screen, the user sees a large, prominent "+1 Kafe" button.
- **Step 2:** (Optional) User toggles coffee type using emoji buttons (☕️ Espresso, 🥛 Macchiato, 🧊 Freddo) placed directly below the main button.
- **Step 3:** User taps "+1 Kafe". If no toggle was selected, it defaults to a standard Kafe.
- **Step 4:** A quick confirmation animation plays.
- **Step 5:** (Optional) User taps an "Add Details" button to open a modal where they can add location, photo, or notes before saving.
- **Step 6:** The Kafe is saved to the database.

## 3. Viewing the Feed
- **Step 1:** User taps the "Feed" tab in the bottom navigation.
- **Step 2:** User scrolls through a chronological timeline of recent Kafes logged by all 8 volunteers.
- **Step 3:** Each feed item displays the user's name, coffee type, time, and any optional details/photos.

## 4. Checking the Leaderboard
- **Step 1:** User taps the "Leaderboard" tab in the bottom navigation.
- **Step 2:** User views a ranked list of all 8 volunteers based on the total number of coffees consumed.
- **Step 3:** The leaderboard highlights the top coffee drinker(s).
