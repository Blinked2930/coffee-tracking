# Product Requirements Document (PRD)
**Project:** Kafe Tracker

## Overview
Kafe Tracker is a simple, high-design, low-friction mobile web app for 8 volunteers to track their coffee dates. The primary goal is to make logging a kafe fun, fast, and satisfying, with a focus on a stellar mobile experience.

## Core Features & User Flow
* **Frictionless Login:** A single-screen login requiring only the user's Name and a PIN (1-8).
* **The Main Action:** A prominent, highly satisfying "+1 Kafe" button on the home screen.
* **Quick Modifiers:** A row of emoji toggles directly under the main button for coffee type (☕️ Espresso, 🥛 Macchiato, 🧊 Freddo) to avoid typing. Defaults to a standard kafe if untouched.
* **Optional Details:** An expandable section or modal to add location, a photo, or notes/friends.
* **Dual Dashboards:** * **Feed Tab:** A chronological timeline of everyone's recent kafes.
    * **Leaderboard Tab:** A ranked list showing who has drank the most coffee.

## Target Audience & Platform
* **Users:** Exactly 8 predefined users. 
* **Design:** Mobile-first, uncluttered UI.
* **Platform:** Progressive Web App (PWA) allowing users to save it directly to their iOS or Android home screens.
